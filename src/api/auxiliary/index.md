# 辅助

辅助性资源负责为 Apps 提供其他支持，或者为用户提供实用功能。下表总结了所有辅助性资源：

| API                                             | 来源        | 作用                     |
| ----------------------------------------------- | ----------- | ------------------------ |
| [Secret](./secret.md)                           | Kubernetes  | 存储敏感信息并用于配置   |
| [ConfigMap](./configmap.md)                     | Kubernetes  | 存储非敏感信息并用于配置 |
