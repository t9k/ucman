安全设置

点击左侧的账户设置 > 安全设置，进入安全设置页面：

安全设置页面提供了两个功能，修改密码和管理 API Key。这里我们只演示管理 API Key。

初次使用时，需要点击管理 API Key 的启用，输入密码来开启 API Key 功能。然后，点击生成 APIKey 进入创建页面：
